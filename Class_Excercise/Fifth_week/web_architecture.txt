웹 컨텐츠 == html + css + java script

1. html : hyper text(링크를 클릭해서 다른 문서나 사이트로 즉시 이동할 수 있는 기능) markup Language
    웹 표준 : 웹 사이트를 만들 때 지켜야 하는 약속들을 정리한 것 , 어떤 브라우저에서 접근해도 깨지지 않도록 하는 것이 목표
            html5 + css3로 문서를 만드는 것이 웹 표준을 지킨 문서를 만드는 기본
    웹 브라우저 : 기본적으로 html 리더 

    기본 문서 구조 
        1) 태그(tag) : 마그업 할 때 사용하는 약속된 표기법
            <와 >를 이용하여 작성
            소문자로 작성
            들여쓰기를 위해 tab키를 사용
            태그는 속성과 함께 사용 가능
            포함 관계를 명확히 해야 함. 
                <!DOCTYPE html> --현재 문서가 HTML5로 작성되었음
                <html lang = "ko"> -- 웹문서의 시작을 알리는 태그
                <head> --문서 정보를 지정하는 태그
                <meta charset="UTF-8"> --문자세트를 비롯한 문서 정보 - 데이터에 대한 데이터 , 웹 문서에 대한 정보지정
                <meta name = "viewport" content = "width-device-width, initial-scale=1.0"> --모바일 기기 고려하는 코드
                <body> --실제 화면에 보이는 문서 내용을 입력하는 태그 
                <h 1~6> 제목 표시 (글씨가 굵고 진하게 표시) (숫자가 작을 수록 글씨 크기가 커짐)
                <p> --paragraph 단락 표시
                <br> --줄 바꾸기
                <hr> -- horizontal line 가로 줄 넣기
                <blockquote> --인용문 표시
                <pre> --preformat 입력 그대로 화면에 표시 
                <strong>,<b> --텍스트를 굵게 표시할 때 사용
                <em>,<i> --이탤릭체로 표시 , em은 특정부분을 강조하고 싶을 때, i는 단순히 이탤릭체
                <mark> 형광펜
                <span> --일부 텍스트만 묶어 스타일 적용
                <ul>,<li> -- 순서가 필요하지 않은 목록 만들기 , 항목 앞에 작은 불릿이 붙음. 중첩 가능
                <ol>,<li> -- 순서가 필요한 목록 만들기 , 중첩 가능 (순서는 자동)
                <dl><dt><dd> -- 제목과 설명이 한 쌍인 목록 만들기 <dl> 은 목록 <dt>는 제목 <dd>는 설명을 표시할
    =====================표관련 태그======================================================
                <table>,<tr>,<td>,<th> 
                    -<table>로 표가 들어갈 자리를 만들고
                    -<tr>로 행을 생성
                    -<td>로 셀을 생성
                    -<th>로 제목 셀 시정
                    -colspan : 가로로 셀을 합침 , -rowspan : 세로로 셀을 합침 (셀 병합)

                <caption> 
                    -<table>다음에 사용되고 ,표에 제목을 붙일 수 있다.
                <figcaption>
                    -<table> 위,아래에 사용되고, 표의 제목을 붙일 수 있다. 
    =======================이미지 ,링크 태그=========================================
                <img>
                    -gif, jpg, png : src 속성을 사용해 이미지 차일 경로를 지정해야만 이미지 표시가 가능
                <a>
                    -텍스트나 이미지 앞에 지정되며, 클릭시 다른 페이지로 넘어갈 수 있도록 하는 태그
                    -속성
                        -href : 링크한 문서나 사이트 주소 입력
                        -target : 링크한 내용이 표시될 위치 지정
                        -download : 링크한 내용을 보여주는 것이 아니라 다운로드 

    =======================폼 관련 태그==================================
                <form>
                    -사용자가 뭔가를 입력하는 형태
                    -기본적 태그
                        -method : 사용자가 입력한 내용들을 서버쪽 프로그램으로 어떻게 넘겨줄지 지정
                            -get : 주소 표시줄에 사용자가 입력한 내용이 그대로 그러남 
                            -post : 입력 내용 길이에 제한을 받지 않고 사용자가 입력한 내용이 드러나지 않음.
                        -name : 폼 이름 지정
                        -action : <form>태그 안의 내용들을 처리해줄 서버 상의 프로그램 지정 
                        -target : action 속성에서 지정한 파일을 지정한 위치에서 열도록 지정
                <label>
                    -폼 요소에 레이블을 붙이기 위한 것 : 폼요소와 레이블 텍스트가 서로 연결되어 있다는 것을 브라우저가 알 수 있음.
                    ex> 아이디, 비밀번호 처럼 입력 참 옆에 붙여 놓는 텍스트

                <fieldset>,<legend>
                    - <fieldset> 태그 사이에 폼들을 하나의 영역으로 묶고,
                    - <legend> 는 위 fieldset의 제목 
                <input>
                    -사용자가 입력하는 부분
                    -type 속성에 사용가능한 유형 
                        -hidden : 사용자에게는 보이지 않지만 서버로 넘겨지는 값
                        -text : 한줄 짜리 텍스트를 입력할 수 있는 텍스트 상자
                        -search : 검색 상자
                        -tel : 전화번호 입력 필드
                        -url : url 주소를 입력할 수 있는 필드
                        -email : 메일 주소를 입력할 수 있는 필드
                        -password : 비밀 번호를 입력할 수 있는 필드
                        -datetime : 국제 표준시로 설정된 날짜와 시간
                        -date : 사용자 지역 기준 날짜 (년,월, 일)
                        -month : 사용자 지역 기준 날짜 (년 , 월)
                        -number : 숫자를 조절할 수 있는 화살표
                        -range : 숫자를 조절할 수 있는 슬라이드  막대 넣음
                        -color : 색상 표
                        -file : 파일 첨부 버튼
                        -submit : 서버 전송 버튼
                        -checkbox : 주어진 항목에서 2개 이상 선택 가능한 체크박스
                        -radio : 주어진 항목에서 1개만 선택 가능한 
                        -autofocus : 페이지에 들어가자마다 첫번째 박스에 커서 깜빡
                        -button : 버튼
                <select> <optgroup> <option> 
                    - 주어진 옵션 중에서 선택할 수 있도록 드롭다운 목록
                <textarea>
                    -여러 줄의 텍스트를 입력할 수 있도록 함

2. css (Cascading Style Sheet)
    -html과 함께 웹 표준의 기본 개념 , 디자이을 구성
    -스타일 형식 : p {text-align : center ; }
            선택자    스타일 속성     속성값
    -스타일 시트 
        -외부 스타일 시트 <head><link rel "stylesheet" href = "css/style.css"></head>
        -인라인 스타일  <span style = "color:darkblue;"> 평가합니다.</span>
    -주요 선택자
        -전체 선택자 : 스타일을 모든 요소에 적용할 때 사용 
            - *{margin : 0; padding : 0;} -> 여백 없음
        -태그 선택자 : 특정 태그가 쓰인 모든 요소에 스타일 적용
            -p {text-align : center;} ->텍스트 단락을 중앙 정렬
            -h2 {color : darkblue ; }
        -클래스 선택자 : 태그 대신 클래스 이름을 선택하여 지정된 클래스에 스타일 적용
            -.reg {font-size : 14px ; color : darkblue ;}
        -id 선택자 : 웹 문서 안의 특정 ㅂ분에 스타일 적용 (문서 안에서 한 번만 적용 가능)
            -#container{width : 600px;(너비) padding : 15px ;(테두리와 내용 사이의 여백) border : 1px dotted gray ;(테두리 스타일) } 
        -그룹 선택자 : 두 개 이상의 요소에 같은 스타일 적용 
            -h1,h2 {text-align : center; }

    =======================글꼴 관련=======================
    -font-family :텍스트를 사용하는 요소들에 사용
        - p{font-family : "맑은 고딕";}
    -@font-face  : 웹 폰트 추가 -> 웹 문서 작성 시 웹 문서 안에 글꼴 정보도 함께 저장해서 접속할 때 사용자 시스템으로 다운로드 시켜서 적용시키는 방식
        - <style>@font-face{src: url();}</style>
    -font-size : 글자 크기 조절
        -em : 해당 글꼴의 대문자 M의 너비를 기준으로 크기 조절 (1em == 16px)
        -ex : 해당 글꼴의 소문자 x의 높이를 기준으로 크기를 조절 
        -px : 모니터에 따라 상대적 크기
        -pt : 일반 문서에서 사용하는 크기
    -font-weight : 글자 굵기 조절 
        -normal : 기본값
        -bold | lighter | bolder : 굵게 / 가늘게 /더 굵게
        -100~900 : 400은 normal 700은 bold
    -font-style : 글자 스타일 지정
        -normal : 기본값
        -italic : 이탤릭체
    -font : font-size , font-style , font-weight , font-family 등 속성을 한 꺼번에 묶어 사용 가능
        -p.txt {font : italic 12px 돋움;}
    -color : 글자 색
        -색상 값은 16진수, rgb(rgba), hsl(hsla) 또는 색상이름으로 표기 가능
            -16진수 ex) #ffff00
            -rgb : rgb(255,0,0)
            -hsl : hsl(360, 100%, 0%) 
                    hue(색상), saturation(채도), lightness(밝기)
    -text-transform : 영문자의 대소문자 변환
        -capitalize 
        -uppercase 
        -lowercase 
        -none 
    =======================문단 관련=======================
    -text-align : 텍스트 정렬 
        -start : 현재 텍스트 줄의 시작 위치에 맞추어 문단을 정렬
        -end : 현재 텍스트 줄의 끝 위치에 맞추어 문단을 정렬
        -left : 왼쪽 정렬
        -right : 오른 쪽 정렬
        -center : 가운데 정렬
        -justify  : 양쪽에 맞추어 정렬
        -match-parent : 부모 요소를 따라 정렬
    -line-height : 문단의 줄 간격 조절 
    -text-overflow 
        -whight-space : nowrap으로 지정해서 줄 바꿈 하지 않을 경우 넘치는 텍스트를 처리하는 것
    -list-style-type : 목록에서 불릿 모양
    =======================배경 관련=======================
    -background-color : 배경 색
    -background-image : 배경에 이미지 삽입
    -background-repeat : 배경에 이미지를 복붙할 건지
        -repeat : 기본 값. 화면에 가득 찰 때까지 가로와 세로로 반복해서 채움
        -repeat-x : 브라우저 창 너비와 같아질 때까지 가로로 반복
        -repeat-y : 브라우저 창 높이와 같아질 때까지 세로로 반복
        -no-repeat : 한 번만 표시하고 반복하지 않음
    - background-size : 배경 이미지 크키 조절 
        - auto : 기본값. 원래 배경 이미지 크기만큼 표시
        - contain : 요소 안에 배경 이미지가 다 들어오도록 확대 혹은 축소 표시
        - cover : 배경 이미지로 모든 요소를 덮도록 이미지를 확대 혹은 축소
    - background-position : 배경 이미지 위치 조절
    -그라데이션 효과
        -linear-gradient : 선형 그라데이션 – 직선 형태로 색상이 바뀌는 것
            to top 아래에서 시작해 위로 그라데이션 생성
            to left 오른쪽에서 시작해 왼쪽으로 그라데이션 생성
            to right 왼쪽에서 시작해 오른쪽으로 그라데이션 생성
            to bottom 위에서 시작해 아래로 그라데이션 생성
        -radial-gradient : 원형 그라데이션 – 원이나 타원의 중심부터 원을 그리면서 바깥으로 색상이 바뀌는 것
        
3. html5와 css3
    -시멘틱 태그 : 태그만 보고도 페이지 구조를 쉽게 이해할 수 있도록 정의된 태그
        -<header> : 특정 부분의 머릿말 
            <nav> : 메뉴 (네비게이터) ->사이트 안의 문서나 다른 사이트의 문서로 연결하는 링크 모음
        -<section> : 컨텐츠를 주제별로 묶을 때 사용 ,<section> 안에 또 다른 <section>을 넣을 수 있음 
            <article> 웹 상에 실제 내용을 넣음, 완전한 하나의 콘텐츠
        -<aside> : 본문 내용 외에 주변에 표시되는 기타 내용
            <iframe> 웹 문서 안에 또 다른 외부 문서를 삽입할 때 사용
                width 인라인 프레임의 너비
                height 인라인 프레임의 높이 
                name 인라인 프레임의 이름
                src 프레임에 표시할 문서의 주소 지정
        -<footer> : 웹 상 제일 아래에 표시되는 태그,<header>, <section>, <article> 등을 모두 사용하면서 푸터 안에 다양한 정보를 넣을 수 있음
            <address> 주로 <footer>태그 안에 사용되며, 웹 페이지 제작자 정보를 넣는데 사용됨

    -css 박스 모델 : 웹 문서에서 내용 배치 시 각 요소를 박스 형태로 구성하는 것
        각 박스 모델은 콘텐츠 영역과 테두리, 여백들로 구성됨. css를 사용한 레이아웃의 기본
        -박스 모델 : 실제 콘텐츠 영역, 박스와 콘텐츠 영역 사이의 여백인 패딩(padding) , 박스의 테두리 (border),박스 모델 사이의 여백인 마진 (margin)이라고 함.
        -블록 레벨 요소 : 태그를 사용해 컨텐츠를 추가했을 때 한 줄을 통째로 차지하는 요소 
                        양 옆으로 다른 요소가 올 수 없음. 패딩, 보더, 마진으로 크기나 위치 지정 가능
                        ex> div , p, hn, ul, ol, form ,table fieldset, address
        -인라인 레벨 요소 : 화면에 표시되는 만큼만 차지하고 한 줄을 차지하지 않는 요소 
                        여러개의 인라인 레벨 표현이 가능
                        ex> img, strong, br, sub , span, input, textarea 
        -display 속성 : 필요에 따라 인라인레벨과 블록 레벨을 변경할 수 있음.
                        1)display : block =>해당 요소를 블록 레벨로 지정
                        2)display : none => 해당요소를 화면에 안 보이게 숨김.
                          visability : hidden => 안 보이는데 화면에 영역은 차지함. (공간 유지)
        -border-style : 박스의 테두리 스타일을 지정 
                        none : 기본 값, 나타나지 않음
                        hidden : 테두리가 나타나지 않음.
                        dashed : 짧은 선으로 표시 
                        dotted : 점선으로 표시 
                        solid : 실선으로 표시 
        -border-color : 박스의 테두리 색상을 지정
        -border-radius : 박스 모서리 둥글게 

    -css 레이아웃 
        -box-sizing : 요소 배치를 위해 요소의 너비 계산에 이용,콘텐츠영역+패딩+테두리+마진까지 합쳐서 width 사이즈 조절 가능
        -float : 웹 요소를 문서 위에 떠 있는 상태로 만듦 , 왼쪽 구석 혹은 오른쪽 구석
                left : 문서 왼쪽에 배치
                right : 문서 오른쪽에 배치
                none : 좌우 어디에도 배치하지 않음. 
        -clear : 한 번 float을 이용해 배치를 결정하면 그 다음에도 넣는 요소도 똑같은 속성이 전달됨 . 그래서 더 이상 유용하지 않다고 알려줄 때 사용
                both : float와 블록 둘 다에게 영향 받겠다. -> 그 자리를 clear 하겠다. (그 시트 뒤에 깔리지 않음)
        -position : 웹 문서안의 요소들을 자유자재로 배치해주는 속성
                    static : 기본값
                    relative : 이전 요소에 자연스럽게 연결해 배치하되 위치 지정가능
                    absolute : 원하는 위치에 지정
                    fixed : 지정한 위치에 고정해 배치 (브라우저 기준)
        -z-index: 한 요소위에 다른 요소 쌓기 위해 순서를 지정, 값이 작을 수록 아래에 쌓이고, 명시하지 않을경우 제일 먼저 삽입된 요소가 1을 가지고 ,그 후 추가되는 요소들은 값이 점점 커지게 됨.
    
    -css 연결 선택자
        -하위 선택자 : 부모요소에 포함된 하위 요소 모두에 스타일이 적용되는 것, 손자 요소 포함
                    ex> #container ul{text-align : center;}
        -자식 선택자 : 부모 바로 밑에 자식에게만 스타일이 적용되는 것
                    ex> #container > ul (text-align: center ;)
        -인접 형제 선택자: 구조상 같은 부모를 가진 형제 요소 
                    ex> h1+ul{text-align : center;} ->h1이랑 가장 가까운 ul한테만 적용
        -형제 선택자 : 모든 형제 요소에 적용
                    ex> h1~ul(text-align: center ; ) h1과 같은 행렬에 있는 ul들에게 적용 
        -속성 선택자 : 태그 안에서 사용하는 속성들의 값에 따라 스타일 지정
                    ex> [target = "_blank"]{text-align : center ; }
                        [target ~= "_blank"]{text-align : center ; }
                        [target |= "_blank"]{text-align : center ; }
                        [target ^= "_blank"]{text-align : center ; }
                        [target $= "_blank"]{text-align : center ; }
                        [target *= "_blank"]{text-align : center ; }
    -css 속성 가상 클래스 
        -가상 클래서 (pseudo class) : 클릭하거나 마우스를 올려놓는 등 특정 동작을 할 때 스타일 적용을 위해 사용
            :link - 아직 방문하지 않은 링크에 스타일 적용
            :visited -한 번이상 방문한 링크에 스타일 적용
            :hover - 마우스 커서를 올려놓았을 때 스타일 적용
            :active - 링크나 이미지를 누르고 있을 때 스타일 적용
            :focus - 포커스가 맞춰졌을 때 스타일적용 (텍스트 박스,검색 창)
4. 반응형 웹 : 접속한 기기에 따라서 레이아웃을 바꿔서 보여주는 디자인 -> 웹 표준
    -뷰포트 (viewport) : 접속한 사용자에게 보여지는 화면 영역 
            <meta name ="viewport" content ="width = device-width,inital-scale = 1">
            @media  미디어 유형 [and 조건] *[and 조건] {CSS Code ; }
                -미디어유형 종류 
                    -all 
                    -print
                    -screen : 컴퓨터 스크린 (스마트폰 포함)
                    -tv 
                    -aural :음성 합성 장치
                    -braile : 점자 표시 장치
                    -handheld : pad 처럼 손에 들고 다니는 장치
                    -projection : 프로젝터 
                    -tty: 디스플레이 기능이 제한된 장치
                    -embossed : 점자 프린터
            ex> @media screen and(max-width : 1024px){
                body{
                    background : url(image/choco.jpg) no-repeat fixed;
                    background-size : cover;
                }
