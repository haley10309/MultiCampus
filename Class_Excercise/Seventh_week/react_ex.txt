

-모던 자바스크립트
    : 리액트, 뷰, 앵귤러 등 가상 dom을 이용하는 라이브러리 혹은 프레임워크를 사용
    npm, yarn 등의 패키지 관리자 사용
    주로 ES2015 이후의 표기법 사용
    웹팩 등의 모듈 핸들러 사용
    바벨 등의 트랜스파일러 사용
    spa 로 작성

    - 트랜스파일러(transpiler)
    :자바 스크립트 표기법을 브라우저에서 실행할 수 있는 형태로 변환해줌
    트랜스파일러를 사용하면 새로운 표기법으로 작성된 자바스크립트를 오래된 표기법으로 변환이 가능해짐
        현재는 바벨(babel)이라 불리는 트랜스파일러가 주로 사용됨.
    
    -SPA (single page application) : html 파일은 하나만 사용(표준) -> 자바스크립트를 이용해 dom을 바꿔서 화면 이동을 구현

    -변수 : var 를 이용한 변수 선언의 문제
          재 선언 가능 재 정의 가능
